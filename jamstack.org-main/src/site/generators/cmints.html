CMintS

CMintS is a CMS and Static Site Generator for single and multi-language website creation.

Learn more about CMintS at https://cmints.io/. It’s an open source project and every related repository can be found in the CMintS GitHub organization.

You can either install CMintS globally as it’s described in the installation documentation or as a local dependency, following the Quick Start tutorial.

Single and multi-language website starters can be found at https://single.cmints.io/ and https://multi.cmints.io/ accordingly.

Features

i18n

Comprehensive internationalization out of the box for layout and translatable page content creation.

Theme

Use EJS, LESS and JS modules to create amazing themes for your website.

Markdown

CMintS uses CommonMark for markdown pages creation, alternatively you can also use EJS or a regular HTML.

TMS integration

Integrate your project with Crowdin and start translating your website right away with the help of the translation community.
