The default pages plugin for Docusaurus. The classic template ships with this plugin with default configurations. This plugin provides creating pages functionality.

Installation

bash npm2yarn npm install --save @docusaurus/plugin-content-pages

If you have installed @docusaurus/preset-classic, you donâ€™t need to install it as a dependency. You can also configure it through the classic preset options instead of doing it like below.
